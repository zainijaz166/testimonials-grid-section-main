# Testimonials Grid Section - Frontend Mentor

This is a solution to the Testimonials Grid Section challenge on Frontend Mentor. It is a fully responsive layout that displays user reviews in a complex grid formation.

## Features

- **CSS Grid Layout:** Used advanced CSS Grid properties to create a masonry-style multi-column layout.
- **Responsive Design:** Optimizes the layout seamlessly for mobile (single column) and desktop (4-column grid).
- **Card Styling:** Distinct background colors and visual elements (like quotation marks) for each testimonial.
- **Semantic HTML:** Built using accessible tags like `<main>` and `<article>` for better SEO.

## Technologies Used

- HTML5
- CSS3 (CSS Grid & Flexbox)
- Git & GitHub
